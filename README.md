# SmartBrain-App-v4

This is the fourth version of **SmartBrain App**. It is a face recognition app.

### To run this project:

1. *Clone this repo,*
2. *Run npm install,*
3. *Run npm start,*
4. *Also, clone and run **[SmartBrain-backend-API-v4](https://github.com/MalihaKabir/SmartBrain-Backend-API-v4)** to connect this app with its backend part.*
